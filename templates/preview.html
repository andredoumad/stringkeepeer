Stringkeeper web harvest service automatically extracts the type of information from the internet that you're interested in and stores it into a spreadsheet.

Web harvest solves the following problem:

Search engines don’t find or store all the content on a web page; they simply link you to the content’s location.

The harvest process executes searches then extracts all of the text content from each of the results pages and uses natural language processing to collate for further analysis based on your harvest configuration. Many times you may not be sure where the target data is on the internet, web harvest eliminates hours of manual web searching and data mining. With web harvest you can describe what phrases and keywords interest you and which do not. Then you define what type of data you are looking to find in the spreadsheet. If you're interested in finding someone or something but you're not sure where it might be, this service is for you. You do not need to supply anything beyond a set of descriptions of what it is you're interested in tracking. The harvest process requires little effort for you. Simply describe what phrases describe your target, define a list of phrases you're not interested in and set the data type that you wish to target. There are many filters and options for you to use to better locate what it is you wanted to find.
